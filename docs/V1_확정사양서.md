# V1 확정 사양서 — 유저 플로우 최종 고정판

> 코딩 시작 전 **완전 고정된 설계.** 만들다가 헷갈리면 이 문서가 기준.
> **확정 스케치 = FINAL 붙은 30장 전부** (docs/sketches/). 화면은 스케치와 "아예 똑같이"(픽셀 기준).
> 코스 흐름 9 (홈·1·1.5·2·3·4×3모드·5) + 탭 3 (코스·즐겨찾기·내정보) + 지역관리
> 팝업·시트 9 (저장·어느코스에?·직접추가·지역추가·마이노트입력·삭제확인·즐겨찾기담기·장소추가선택·날짜시간변경·메모)
> 기타 7 (상세·공유·동선지도·빈홈·1.5빈상태·로딩·검색없음)

---

## 전체 흐름

```
홈 ─[＋새 코스]→ 화면1(날짜·시간) → 화면1.5(지역) → 화면2(장소) →[다음]→
화면3(맛집) →[코스 완성]→ 화면4 조립 모드 → (화면5 호텔, 선택) →
[저장하기] → 이름 팝업 → 홈 (예정 코스에 등장)

홈/코스탭 코스 카드 → 화면4 보기 모드 → ✏️ → 편집 모드 → 저장 → 보기 모드
뒤로(<) = 한 단계씩 후퇴 (담은 것 유지). 홈까지 나가면 자동 임시저장 (팝업 없음)
```

## 공통 규칙 (전 화면)

- **담기 [+] = 토글** (한 번 더 누르면 빼기)
- **☆ 찜 = 카드 오른쪽 위, 토글** (확인 없음). 화면 2·3·5·상세 어디서든
- **🗑️ 삭제 = 항상 "삭제할까요?" 확인 팝업 1번** (★해제만 예외 — 토글)
- **카드 탭 = 상세 화면** (화면 2·3·즐겨찾기·화면4 보기 모드에서 동일)
- **[⭐ 즐겨찾기에서 담기]** = 화면 2·3 공통. 시트가 올라옴 → [+] 토글로 여러 개 담기 → [닫기]
- 추가/수정 팝업은 **같은 팝업 재활용** (빈칸=추가, 값 채워짐=수정)
- **팝업 규칙**: 입력·확인 계열 = 가운데 팝업 / 목록 선택 계열 = 아래 시트. **예외: 저장 팝업만 시트** (코스 요약을 보여주는 성격이라)
- **달력 = 공용 부품**: 월 이동(‹ 2026년 7월 ›) + 월 전체 표시. 화면1과 [📅 날짜·시간 변경] 팝업이 같은 부품 공유
- **호텔 표시 (화면4 전 모드)**: 타임라인 안이 아니라 **타임라인 아래 별도 슬롯** ("🏨 호텔 — 코스 밖 별도") — 데이터 설계(hotel은 items 밖)와 일치
- **화면2 여친정보 배너 ▾** = 탭하면 전체 목록 펼침 (다시 탭 = 접기)

## 임시저장 (drafts)

- 코스 만들다 홈으로 나가면 **자동 임시 보관** (어느 화면에서든, 팝업 없음)
- **여러 개 허용** (drafts 배열)
- **홈 예정 코스 맨 위에만** 점선 카드로 표시 (코스 탭엔 없음)
- 점선 카드 탭 = 나갔던 화면부터 재개 / ✕ = 확인 후 그것만 삭제
- [＋새 코스] = 항상 그냥 새로 시작 (묻지 않음)
- [저장하기] 완료 = drafts에서 제거 → courses로

## 화면별 확정 사항

### 화면 0 — 홈
- Header / 마이노트 / [＋새 코스] / 예정 코스(임시본 점선 포함) / 탭바
- 마이노트: 🎂기념일·✅할일·💝여친정보 **각 영역 [+]** → 입력 팝업
- 항목 평소엔 깔끔 → **탭하면 그 항목에만 ✏️🗑️** 노출
- 할일 ☐ 탭 = done 토글 (완료 = ☑핑크 + 취소선)
- **기념일**: dday 저장 안 함(date로 계산). **repeat=true → 지나면 내년 D-로 / false → 지나면 자동 삭제.** 🔁매년 뱃지 표시
- 예정 코스 카드에 🗑️ 없음 (코스 삭제는 편집 모드에서만). 홈=예정만, 다녀온은 코스 탭

### 화면 1 — 날짜·시간
- 날짜: 달력 톡 (필수) / 시간: 휠, 기본 12:00~23:00 (필수 아님)
- **[다음] = 날짜 골라야 활성화** (회색→핑크)
- 뒤로 = 홈 (뭔가 입력했으면 자동 임시저장)

### 화면 1.5 — 지역
- 카드 = **내정보 탭 "내 지역 관리"에 추가한 현만**
- 탭 = 선택 표시 → **[다음] 눌러야 진행**
- 검색창 = 목록에 없는 현 검색 (**1회용**, 내 목록에 안 쌓임)
- 첫 실행(0개): 빈 화면 + "검색해서 골라줘"

### 화면 2 — 장소 (볼거리·체험)
- 검색창(통째 검색) / 여친정보 배너 / 지역 탭 / 카테고리 / 가격 / 정렬
- [+담기] 토글 / ☆찜 / 카드 탭=상세(거기서도 담기) / [⭐즐겨찾기에서 담기]
- **[다음] = 0곳도 통과**

### 화면 3 — 맛집
- 상단: 담은 장소 탭(탭=지도 중심 이동) + 검색창
- **처음: 지도 깨끗 + 카테고리 버튼만.** 카테고리 = **토글 + 동시 여러 개 ON** → 켠 것만 핀+목록
- 가격 필터 / [건너뛰기] / 담기·☆·상세·즐겨찾기 시트 = 화면2와 동일
- [코스 완성] → 화면4 조립 모드

### 화면 4 — 3모드 (같은 화면, mode state)
- **개별 시간 없음** (필요하면 메모에 "13시쯤"). **이동 카드 없음.** 시간축(화면1의 시작~끝)은 배경 눈금
- **조립 모드** (새 코스 흐름): 🧺대기 영역(담은 카드) + 빈 타임라인 → **드래그로 배치** (드래그 막히면 임시 ▲▼ 탈출구). **미배치째 저장 가능.** [호텔 추가][저장하기]. 편집 기능 없음
- **보기 모드** (홈/코스탭에서): 읽기 전용. 카드 탭=상세. [🗺️지도]=동선 지도(핀+순서 선)+[구글맵으로 열기]. [📤공유]. 우상단 ✏️
- **편집 모드** (✏️): 조립과 같은 조작 + [＋장소 추가](볼거리/맛집 선택 팝업→화면2/3, 담으면 대기 영역으로) + 카드 ✕빼기 + 카드 탭=메모 수정 + [📅 날짜·시간 변경] + [🗑️ 이 코스 삭제](확인) + [저장]→보기 모드

### 화면 5 — 호텔 (선택)
- 마지막 배치 장소가 지도 중심. **열자마자 핀+목록** (카테고리 단계 없음)
- **[🏨 호텔][💜 러브호텔] 탭** — 호텔=type 검색(정확) / 러브호텔=키워드 "ラブホテル" 검색(가끔 일반 호텔 섞일 수 있음, 한계 인지하고 채택)
- 검색창 / ☆찜 / **[선택] 방식 — 하나만, 다른 거 누르면 교체** / [건너뛰기]

### 저장 팝업
- 이름 **빈칸, 자유 입력** (자동 제안 없음) → [저장] → 홈으로

### 코스 탭
- [예정][다녀온] (date로 자동 분류). 임시본 없음 (홈에만)
- 카드: **이름·날짜·장소수·📍지역** (호텔 아이콘 없음)
- 카드 탭 → 화면4 보기 모드. **다녀온 코스도 ✏️ 편집 가능**

### 즐겨찾기 탭
- 상단 필터 **[장소][맛집][호텔]**
- ★ 탭 = 바로 해제 (토글) / 카드 탭 = 상세
- [+코스에 추가] → "어느 코스에?" 팝업 (**예정 + 임시본** 목록) → 그 코스의 대기 영역으로

### 내정보 탭
- 💑 커플 정보 (사귄 날 → D+표시)
- 📍 **내 지역 관리** (현 목록 + 🗑️확인 삭제 + [＋추가]→검색 팝업)
- 🗑️ 데이터 초기화 (**확인 2번**)
- 앱 정보

### 상세 화면 (공용 — 화면2·3·5·즐겨찾기·화면4보기에서 재활용)
- 큰 사진 / 이름·⭐·가격·영업시간 / 📍위치
- **[🌐 공식 사이트]** (website — 예약은 여기로 연결. 없는 장소면 버튼 숨김)
- **[🗺️ 구글맵에서 보기]** (url — 리뷰 등)
- ☆찜 / [+담기 or 선택] (연 맥락에 따라) / 메모 보기·수정
- 직접 예약(앱 안 완결)은 V2

### 공유 (보기 모드 [📤])
- **텍스트 공유**: 코스를 글로 정리 + **구글맵 코스 링크** (경유지 경로 URL — 받은 사람이 탭하면 구글맵에 동선 열림)
- 미리보기 팝업 → [복사] [LINE으로]
- 이미지 공유는 V2

## 데이터 변경 사항 (설계도 반영 필요)

- `CourseItem`: startTime/endTime **삭제** → { place, memo } (순서는 items 배열 순서)

- `Place`: 구글에서 website, url 필드도 받기
